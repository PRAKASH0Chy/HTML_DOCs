What is a Nested List?

A nested list in HTML is a list within another list. This structure allows you to create a hierarchy of items.

Usages:

  • Representing Complex Data
  • Subcategories
  • Categories

How it can be Created: Nested lists can be created using both ordered lists <ol> and unordered lists <ul>

Example: Nested Ordered List →

Explanation:

  • The outer <ul> creates a top-level unordered list.
  • Inside this list, the <li> items represent different categories (Fruits and Vegetables).
  • The nested <ul> elements within these <li> items represent subcategories.
