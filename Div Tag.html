DIV TAG <div> →

The <div> element is used as a container for other HTML elements.

Key Points:

  • The <div> element is by default a block element
  • meaning that it takes all available width, and comes with line breaks before and after.

Example:

Purpose of <div> tag →

  • Organizes the content of a webpage.
  • Groups elements logically especially when building layouts.
  • Acts as a wrapper for sections of the webpage.
