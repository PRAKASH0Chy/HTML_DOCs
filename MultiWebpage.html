Multi-pages →

A Multi-Page Website is a website made of many separate HTML pages connected together.

As, you see this Where we click on link it will let you land in new page.
This is an example of Multi-page

here we use Anchor tag.

If you don't know about Anchor tag → Anchor Tag

Example:
  • index.html(Home page)
  • about.html(About us)
  • Contact.html(Contact us)

How it works?

You can click links in the menu:
Home → about.html → contact.html → back to index.html Each page loads as a new HTML document.
