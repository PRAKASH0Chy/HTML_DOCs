What are Form Elements?

Form elements are the building blocks inside a form where users can type, select, or upload information.

Common Form Elements:

Element What It Does?
<input> Allows user to type text, select options, etc.
<textarea> Multi-line text input for longer messages.
<button> Button to submit the form.
<select> Drop-down list where user selects an option.
<label> Describes what input means for the user.
<fieldset> Groups related form elements together
<legend> Title for a group of elements in fieldset.

Buttons →

Buttons help users perform actions inside a form.

Button Type What it Does
Submit Submits the form data to the server.
reset Resets all form fields back to their defaults.
button Custom button (does not submit/reset by itself; used with JavaScript).
