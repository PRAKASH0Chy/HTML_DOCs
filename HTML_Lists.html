HTMl Lists →

Well why we need List?

→ lists allow web developers to group a set of related items in lists.

1. Unordered List

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.

Key point: Under unordered list → The list items will be marked with bullets.

Ex:
<ul>
<li>Coffee<li>
<li>Tea<li>
<li>Milk<li>
<ul>

2. Ordered List

An Ordered list starts with the <ol> tag. Each list item starts with the <li> tag.

Key point: Under Ordered list → The list items will be marked with Numbers in Sequential order.

Ex:
<ol>
<li>Coffee<li>
<li>Tea<li>
<li>Milk<li>
<ol>
