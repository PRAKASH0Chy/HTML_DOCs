Table Structure:

<table>, <tr>, <th>, and <td> Tags HTML tables are built using a few key tags:

  • <table>: Defines the table.
  • <tr>: Represents a row in the table.
  • <th>: Defines a header cell (bold and centered by default).
  • <td>: Defines a standard data cell.

Everything between <td> is the content of a table cell

HTML Table Tags

Tag Description
<table> Defines a table
<th> Defines a header cell in a table
<tr> Defines a row in a table
<td> Defines a cell in a table
<thead> Groups the header content in a table
<tbody> Groups the body content in a table

Colspan →

If you want a table cell to stretch horizontally across multiple columns, use the colspan attribute.

What Happens:

  • The cell with "Fruits" uses rowspan="2".
  • So "Fruits" will stretch across 2 rows vertically.

Rowspan →

If you want a table cell to stretch vertically across several rows, you use the rowspan attribute

What Happens:

  • The "Details" header uses colspan="2".
  • So "Details" stretches across 2 columns horizontally.
