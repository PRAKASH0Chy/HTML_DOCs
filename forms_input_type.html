Forms → Input_Types:

Actually what is the need of Input -Types:

  • If you use <input type="email">, the browser shows a special keyboard on mobile that includes the “@” symbol.
  • If you use <input type="date">, the browser shows a calendar to select a dat
  • For Proper Data Collection.
  • Makes form easy, safe, and mobile-friendly →
  • Prevents mistakes automatically.

The <input> tag can have many types, each for different purposes:

Type Purpose
text Single-line text input
password Input for passwords (text hidden)
email Input for email address
number Input for numbers
checkbox Input for numbers
radio Input for numbers
file Upload a file
submit Button to submit the form
date Choose a date
range Select a number by sliding a range
