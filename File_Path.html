HTML → File-paths:

A file path describes the location of a file in a web site's folder structure.

Types of File Path :→

  • Absolute File Path
  • Relative File Path

Absolute File Path →

An Absolute Path is nothing but a full URL to a file.

Example: When we use image tag.
we put full address URL of that particular image(pick from sites)

<img src = "URL" alt="Alternative name" >
