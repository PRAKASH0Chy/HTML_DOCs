Elements Description(What is does)
<!DOCTYPE html> Defines that this is an HTML5 document.
<html lang="en"> The root element of the webpage. lang="en" means language is English.
<head> Contains metadata (information about the page, not shown on screen).
<meta charset="UTF-8"> Defines character encoding (UTF-8 covers most characters worldwide).
<meta name="viewport" content="width=device-width, initial-scale=1.0">& Makes the page responsive on all devices (mobile, tablet, desktop).
<title> The title shown in the browser tab.
<body> The visible content of the web page.

Why Boilerplate is Important →

  • Provides a solid structure to start with.
  • Ensures correct rendering across devices (via viewport meta tag).
  • Prevents common mistakes like missing DOCTYPE.
  • Easy to extend with CSS & JavaScript later.
