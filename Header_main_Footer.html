Header →

The <header> is used to define the top part of a webpage or section.

Usually contains:

  • Website logo
  • Navigation menu
  • Navigation menu

Main →

The <main> tag holds the main content of the webpage.

Usually contains:

  • Only one <main> tag should be used per page.
  • Contains things like articles, text, images, videos, etc.

Footer →

The <footer> tag is used to define the bottom part of the webpage or section.

Usually contains:

  • Contact info
  • Copyright text
  • Links to privacy policy, terms, etc.

Conclusion:

  • Header → Top of page
  • Main → Important content
  • Footer → Bottom of page
