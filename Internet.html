Internet →

The Internet is a network that connects millions of computers and devices all around the world.
It allows them to share information (websites, emails, videos, files) easily.

How Data is Transferred? →

Steps are:

  1. You type a website (like www.google.com ) into your browser
  2. Your computer sends a request to the server where that website lives.
  3. The server processes your request and sends back the data.
  4. Data is broken into small packets that travel through many computers (routers) over cables or wireless signals.
  5. Packets arrive and your browser reconstructs them into the full webpage.

IP Address →

An IP Address is a unique number assigned to every device on the internet (computers, phones, websites).

Example: (IPv4): 192.168.1.1

It’s like a home address but for your computer or website.


Don't know About IPv4 → IPv4

Access a Website →

Steps are:
  1. You type a domain name (e.g., www.google.com ).
  2. Your computer asks a DNS (Domain Name System):
    • “What is the IP address of www.google.com?”
  3. The DNS replies:
    • “It’s 142.250.72.196”
  4. Your computer sends data packets to that IP address.
  5. Now, you can see the webpage in your browser.

Who Assigns the IP Address →

IP Address are assigned by:

  • Internet Service Providers (ISP).
  • Global organizations like IANA (Internet Assigned Numbers Authority).
  • For local networks, your router automatically assigns IPs

MAC Address →

A MAC Address (Media Access Control Address) is a unique identifier for a device’s network card.

Example: 00:1A:2B:3C:4D:5E

  • It’s used for local network communication (like your home Wi-Fi).
  • Fixed by the manufacturer, unlike IP addresses.

Port Number →

A Port Number helps your computer manage multiple services at once.

Example: Web server → Port 80 (HTTP) or 443 (HTTPS)

Quick concept: Think of your computer as an apartment building (IP address) and port numbers as apartment numbers (specific services).

LAN →

A LAN (Local Area Network) is a small network that connects computers and devices in a close area — like your home, school, or office.

Example: Your home Wi-Fi connects your phone, laptop, and printer.

Switch →

A Switch is a device that connects multiple devices in a LAN and helps them talk to each other.

Purpose of Switch:

  • Connects computers, printers, etc., in a local network.
  • Sends data only to the intended device (efficient).

Router →

A Router connects your local network (LAN) to the Internet.

Modern Router →

A Modern Router is an advanced device combining:

  • Switch
  • Wi-Fi access point
  • FireWall
  • DHCP Server
